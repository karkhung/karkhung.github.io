## Accessing the Admin Panel

1. Navigate to `/admin` on your website (e.g., `http://localhost:4000/admin`)
2. Authenticate with your credentials
3. You'll see the admin dashboard with different content sections

## Creating Blog Posts

1. From the admin dashboard, click on "Posts"
2. Click "New Post"
3. Fill in the required fields:
   - **Title**: The post title
   - **Date**: Publication date
   - **Layout**: Select the appropriate layout (usually `post`)
   - **Body**: Your post content in Markdown
   - **Excerpt**: A short summary of the post
   - **Categories/Tags**: Add relevant categories and tags
4. Add a featured image (if applicable)
5. Click "Save" to save as draft or "Publish" to make it live

## Managing Audiobooks

1. From the admin dashboard, click on "Audiobooks"
2. To add a new audiobook:
   - Click "New Audiobook"
   - Fill in the audiobook details (title, author, description)
   - Upload cover image
   - Add chapters with audio files
3. To edit an existing audiobook:
   - Click on the audiobook title
   - Make your changes
   - Click "Save" or "Publish"

## Working with Pages

1. From the admin dashboard, click on "Pages"
2. To create a new page:
   - Click "New Page"
   - Enter the page title and URL path
   - Select a layout
   - Add content using the editor
3. To edit an existing page:
   - Click on the page title
   - Make your changes
   - Click "Save" or "Publish"

## Media Management

1. From the admin dashboard, click on "Media"
2. To upload files:
   - Click "Upload"
   - Select files from your computer
   - Files will be uploaded to the media library
3. To use media in your content:
   - While editing, click the media button in the editor
   - Select or upload the desired file
   - Insert it into your content

## Tips for Better Content

- Use descriptive file names for images and media
- Optimize images before uploading (recommended max width: 1200px)
- Use headings (H2, H3) to structure your content
- Add alt text to images for accessibility
- Preview your changes before publishing
